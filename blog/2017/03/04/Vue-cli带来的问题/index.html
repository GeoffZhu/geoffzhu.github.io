Vue-cli带来的问题

用不就行了，干嘛想那么多。，，，，我：鄙视脸 (¬､¬)

其实事情的起因是这样的，招到了一个奇葩的同学，聊天中感觉应该是个很厉害的前端，聊到的东西包括，vue全家桶， 代码规范，webpack等等，感觉逻辑也不错，就要了。

后来，他问过我的问题如下：
我npm run dev 怎么没有弹出页面？
密码校验是不是应该在前端做？

然后我看了他的开发流，连chrome 的Devtools都不会用啊啊啊！看请求都是我手指着那个Network的tab，他才知道有这个东西！其实这些我都能忍，新人，要教。但是你不要在我手指着chrome的network标签时说”这个啊！我知道”，我已经吐血了。

各种脚手架让配置轻松了不少，但别在它们中迷了路。我不是啥大牛，但我知道 “认真做事，步子大了，真能扯到蛋！”

越来越相信 Talk is cheap, show me the code， 完。
